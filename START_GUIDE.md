# Инструкция по запуску проекта DogovorAI

Этот документ содержит все необходимые команды для самостоятельного запуска веб-приложения DogovorAI на вашем устройстве.

## Шаг 1. Откройте терминал
Убедитесь, что вы находитесь в корневой папке проекта (`/Users/damikosya/Startapp_DogovorAI`).

## Шаг 2. Активируйте виртуальное окружение
Прежде чем запускать проект, необходимо запустить виртуальную среду Python, в которой установлены все библиотеки нашего проекта. Выполните команду:
```bash
source .venv/bin/activate
```
*(Вы поймете, что оно активировалось, если в терминале слева появится надпись `(.venv)`).*

## Шаг 3. Запуск сервера FastAPI
Теперь запустите сам бэкенд-сервер командой:
```bash
uvicorn main:app --reload
```
Флаг `--reload` означает, что сервер будет автоматически перезагружаться, если вы внесете какие-либо изменения в файлы кода.

## Шаг 4. Откройте сайт в браузере
Когда сервер запустится, вы увидите сообщение в терминале `Uvicorn running on http://127.0.0.1:8000 (Press CTRL+C to quit)`.
Перейдите по одной из этих ссылок:
* **Основной сайт приложения:** [http://localhost:8000/app](http://localhost:8000/app)
* **Документация разработчика (API Swagger):** [http://localhost:8000/docs](http://localhost:8000/docs)

---
 
## 💳 (Опционально) Запуск тестирования оплат Stripe
Если вам нужно тестировать покупки тарифов (Pro/Max), нужно запустить слушатель `webhook` для Stripe в отдельном, втором окне терминала!

Откройте **новую вкладку терминала**, перейдите в папку проекта и введите:
```bash
stripe listen --forward-to localhost:8000/api/subscriptions/webhook
```
Это позволит Stripe сообщать вашему локальному серверу о том, что оплата прошла успешно, и тариф в системе поменяется на Pro или Max.

## 🛑 Как остановить сервер?
Чтобы выключить сервер, просто нажмите комбинацию **Ctrl + C** в терминале, где он запущен.
Если захотите выйти из виртуальной среды, используйте команду `deactivate`.
