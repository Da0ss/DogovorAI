# Проектный план DogovorAI

## Этап 1: MVP (Текущий этап) 
- [ ] Инициализация проекта (FastAPI + Supabase интеграция)
- [ ] Модуль обработки файлов: поддержка PDF/DOCX и OCR для фото
- [ ] Интеграция Kimi API: промпт-инжиниринг для поиска юридических рисков
- [ ] База знаний: механизм сопоставления текста с официальными законами
- [ ] Базовый Web-интерфейс: форма загрузки и окно с результатами анализа

## Этап 2: Монетизация и Масштабирование 
- [ ] Система подписок: 3 уровня доступа (Free, Pro, Business)
- [ ] Интеграция платежных систем через Supabase
- [ ] Разработка личного кабинета пользователя для хранения истории проверок

## Этап 3: Human-in-the-loop 🤝
- [ ] Интерфейс для "Дежурных юристов"
- [ ] Система тикетов: передача сложного документа на проверку эксперту
- [ ] Подготовка к миграции на мобильное приложение (API-first подход).
